## SPEKTR aka ЭМВ "Спектр" aka Электро магнитная винтовка "Спектр"
### Про проект:
Электромагнитная пятиступенчатая(пятикатушечная) винтовка. В качестве питания используются 4 аккумулятора 18650 соединенных последовательно. Конденсаторный блок состоит из  пяти конденсаторов, 5 х 1000 мкФ, заряжаемых до 350 В. Управление винтовкой осуществляется с помощью микроконтроллера Arduino Nano, который координирует последовательное включение катушек и обработку других данных

**Подробная информация по проекту появится после его полной сборки которая будет не скоро**
### Про файл "расчёты FEMM.rar"
Архив содержит файлы параметров и результатов расчётов программы FEMM. Файлы БЕЗ приставки "V = (большое число)" являються параметрами расчёта, файлы С приставкой "V = (большое число)" являються результатами расчёта. Подробнее про расчёт Гаусс-пушки в FEMM можно почитать [тут](https://habr.com/ru/sandbox/210428/)

***В результатах всех 5 расчётов длинна провода катушки будет равна примерно 8 см, это БАГ программы, на деле на одну катушку уйдет около 1 м провода!!!***
