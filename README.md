# SPEKTR aka ЭМВ "Спектр" aka Электро магнитная винтовка "Спектр"
Электромагнитная пятиступенчатая(пятикатушечная) винтовка. В качестве питания используются 4 аккумулятора 18650 соединенных последовательно. Конденсаторный блок состоит из  пяти конденсаторов, 4 х 330 мкФ + 1 х 1000 мкФ, заряжаемых до 350 В. Управление винтовкой осуществляется с помощью микроконтроллера Arduino Nano, который координирует последовательное включение катушек и обработку других данных

Для управления используется кнопка спуска и подстроечные резисторы для настройки. Все компоненты размещены в пластиковых корпусах, обеспечивающих компактность и защиту внутренней электроники
