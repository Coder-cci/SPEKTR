## SPEKTR aka ЭМВ "Спектр" aka Электромагнитная винтовка "Спектр"
### Про проект:
Электромагнитная пятиступенчатая(пятикатушечная) винтовка. В качестве питания используются 4 аккумулятора типа 18650. Конденсаторный блок состоит из  пяти конденсаторов, 5 х 1000 мкФ 400 В, заряжаемых до 380 В. Управление винтовкой осуществляется с помощью микроконтроллера Arduino Nano, который координирует работу винтовки

**Подробная информация по проекту появится после его полной сборки которая будет не скоро**
### Про файл "расчёты FEMM.rar"
Архив содержит файлы параметров и результатов расчётов программы FEMM. Файлы БЕЗ приставки "V = (большое число)" являються параметрами расчёта, файлы С приставкой "V = (большое число)" являються результатами расчёта. Подробнее про расчёт Гаусс-пушки в FEMM можно почитать [тут](https://habr.com/ru/sandbox/210428/)

***В результатах всех 5 расчётов длинна провода катушки будет равна примерно 8 см, это БАГ программы, на деле на одну катушку уйдет около 1-2 м провода!!!***
